# Skill S3 — Neurodivergenz-Standard

> *Nicht "für Neurodivergente". Sondern: so designed, dass es allen gut tut.*

## Zweck
Curio World soll von **allen Kindern 9–16** bespielbar sein — mit oder ohne ADHS, Autismus-Spektrum, Dyslexie, sensorische Empfindlichkeiten, motorische Unterschiede.

**Premium-Anspruch:** Nicht "geht für alle", sondern **"exzellent für alle"**. WCAG 2.2 AAA wo möglich. Neurodivergenz-Experten reviewen jede Änderung. Volle Definition: AGENTS.md §11.

## Wann anwenden
- **Bei:** jeder Quest, jedem NPC-Dialog, jeder Animation, jedem Sound, jeder UI-Komponente, jedem Schriftgrad
- **Loop:** B (Produkt)
- **Sub-Agent-Rolle:** Style-Guardian + Ethics-Guardian (Doppel-Check)

## Die 5 Domänen

### Domäne 1 — SENSORIK

```
☐ Helligkeit / Kontrast: in Echtzeit regelbar (Settings-Menü)
☐ Soundeffekte-Lautstärke getrennt von Musik
☐ Animationen reduzierbar (statische Effekte als Alternative)
☐ KEINE plötzlichen Blitzlicht-Effekte — oder mit Vorwarnung (3-Sek-Warnung)
☐ Vibration an/aus (Mobile + Controller)
☐ Farbfilter für Farbenblindheit (Deuteranopie, Protanopie, Tritanopie)
☐ Sanfte Übergänge statt harter Szenen-Wechsel
```

### Domäne 2 — KOGNITIV

```
☐ Quest-Texte: kurz, eindeutig, kein Sarkasmus, keine Doppeldeutigkeit
☐ Audio-Hinweise parallel zu Text (Vorlesen-Funktion)
☐ Icon-Sprache unterstützt Text (manche denken in Bildern)
☐ Pacing-Kontrolle: Spieler kann Schwierigkeit NICHT nur senken,
   sondern auch Geschwindigkeit (Boss-Geschwindigkeit, Timer-Länge)
☐ Speicherbare Routen: Auto-Map-Markierung zu aktuellen Quests
☐ KEINE Strafe für Verirren, Verlieren, Ausprobieren
☐ Optionale Hinweise in 3 Stufen:
   - Stufe 0: kein Tipp
   - Stufe 1: sanfter Hinweis ("Versuch's mal mit...")
   - Stufe 2: direkter Hinweis ("Klick auf das blaue Ding")
☐ "Ruhige Zeiten": Tageszeitfenster mit weniger NPCs, weniger Events
☐ Sensory-Zones: Chill-Bereiche ohne Quest-Druck (Parks, Cafés)
☐ Vorhersehbarer Input: Auto-Sprint, Aim-Assist, Snap-to-Target
```

### Domäne 3 — SOZIAL

```
☐ Solo komplett spielbar (kein "du brauchst Crew-Mate")
☐ Sozial-Distanz-Option: eigener privater Server zum Üben
☐ Mikro aus, Text an als Default
☐ Freundschafts-Anfragen brauchen Bestätigung
☐ Anti-Mobbing-System (3 Verwarnungen = 7-Tage-Sperre)
☐ "Sichere Lobby" für neurodivergente Spieler (gekennzeichnet)
☐ Mentor-System: erfahrene Spieler helfen Neuen (freiwillig, mit Badge)
☐ Keine Vergleichs-Leaderboards außer Opt-In
```

### Domäne 4 — MOTORIK

```
☐ Eingabe-Wahl: Touch, Maus+Tastatur, Controller alle unterstützt
☘� Tasten-Re-Mapping
☐ Vergrößerbare UI (150%, 200%)
☐ Ziel-Hilfen (Snap, Aim-Assist, Vergrößerung beim Zielen)
☐ Pausen-Tolerant: 5-Min-Pause möglich ohne Verlust
☐ Keine Quick-Time-Events unter 800ms (zu schnell für motorische Limitationen)
```

### Domäne 5 — KOMMUNIKATION

```
☐ Schriftart-Wahl: Standard + OpenDyslexic + serif-Option
☐ Schriftgröße: 3 Stufen (klein, mittel, groß)
☐ Kontrast: 3 Stufen (Standard, hoch, sehr hoch)
☐ Untertitel für alle gesprochenen Dialoge
☐ Vorlese-Funktion (TTS) für Texte und Quest-Beschreibungen
☐ Chat-Optionen: nur Emojis, kurze Sätze, voller Text
☐ Keine Rechtschreib-Korrektur beim Tippen (würde Schüler stressen)
```

## Häufige Fehler

| Fehler | Warum problematisch |
|---|---|
| Timer mit < 5 Sekunden | Motorisch+motorisch unmöglich |
| Nur ein Eingabegerät unterstützt | Schließt Kinder aus |
| Soundeffekte ohne Lautstärke-Regelung | Trigger für sensorische Überlastung |
| Quests nur mit Crew spielbar | Sozialer Druck → Ausschluss |
| Leaderboards für alle sichtbar | Trigger für Selbstwert-Probleme |
| Helles Blitzlicht ohne Warnung | Photosensitiv-Epilepsie-Risiko |
