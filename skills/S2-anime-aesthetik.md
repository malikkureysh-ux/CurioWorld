# Skill S2 — Anime-Ästhetik

> Referenzen: Ghibli, Genshin Impact, Pokémon, Animal Crossing — aber kindlicher und niemals creepy.

## Zweck
Stellt sicher, dass **alle Assets, Charaktere und Welten** dem visuellen Nordstern von Curio World entsprechen. Konsistenz ist King — ein einziger "falscher" Asset zerstört das Gefühl.

**Premium-Anspruch:** AAA-Game-Studio-Polish-Niveau. Vergleiche: Genshin Impact, Animal Crossing, Ghibli-Spielfilme. Nicht "anime-y enough", sondern "sieht aus wie ein 50-Millionen-Dollar-Titel". Volle Definition: AGENTS.md §11.

## Wann anwenden
- **Vor:** jedem Blender-Modell, jeder Textur, jedem NPC, jeder Stadt
- **Loop:** B (Produkt)
- **Sub-Agent-Rolle:** Style-Guardian prüft gegen dieses Skill

## Die 7 Checklisten-Punkte

```
☐ 1. CHIBI-PROPORTIONEN
   Großer Kopf, kleiner Körper. Kopf etwa 1/3 bis 1/2 der Gesamthöhe.
   Bei Kindern: Kopf fast halbkugel-rund. Bei Lumi: noch größer.

☐ 2. AUGEN, NASEN, FORMEN
   Große Augen (mindestens 1/3 des Gesichts).
   Winzige oder keine Nase — oft nur ein kleiner Strich/Punkt.
   Rundliche Körperformen, keine eckigen Muskelberge.
   Weiche Konturen, niemals harte Kanten.

☐ 3. FARBPALETTE
   Pastelltöne als Basis (himmelblau, lavendel, mint, pfirsich, butter-gelb).
   Kräftige Akzente nur bei Lumi, Boss-Glows, Item-Highlights.
   Kein Neon, kein Giftgrün, kein Knallrot auf großen Flächen.
   Test: Wirkt die Szene auf einem 12-Jährigen "süß" oder "aggressiv"?
         Süß = gut.

☐ 4. CEL-SHADING
   Weiche Outlines (1-2 px bei Avatar-Größe).
   Klare Licht-/Schatten-Trennung, keine fotorealistischen Verläufe.
   Specular-Highlights nur als sanfte Glanzpunkte, nie spiegelnd.

☐ 5. DIVERSITÄT SICHTBAR
   Verschiedene Ethnien (Hauttöne von hell-porzellan bis tief-braun).
   Verschiedene Körpertypen (auch korpulente, kleine, große).
   Verschiedene Behinderungen (Rollstuhl, Hörgerät, Prothese, sichtbar).
   Verschiedene Altersrepräsentation innerhalb 9–16.

☐ 6. EMOTIONALER AUSDRUCK
   Gesichter müssen mindestens 6 distinkte Ausdrücke tragen:
   neutral, freudig, traurig, verwirrt, ängstlich, begeistert.
   Lumi: zusätzlich "schlaf", "kuschelig", "stolz".
   NPCs: zusätzlich "schüchtern", "temperamentvoll" je nach Rolle.

☐ 7. KEIN CREEPY, KEIN MATURE
   Keine Horror-Elemente (selbst Halloween-Event: niedlich-gruselig, nie scary).
   Keine Waffen, die wie echte aussehen (Zauberstäbe ja, Pistolen nein).
   Kein sexualisierter Look, egal welcher Charakter.
   Keine politische Symbolik (auch keine ironische).
```

## Stilistische Referenzen (gut für Moodboard)

- **Lumi selbst:** Mischung aus Pikachu (Kulleraugen, Kullerbauch) + Ghibli-Totoro (Wärme, Weichheit) + Genshin-Drache (Stil-Treue)
- **NPCs:** Animal-Crossing-Warm + Ghibli-Detailtiefe
- **Welten:** Genshin-Städte (Vielfalt, Wahrzeichen) + Mario-Odyssey-Farben
- **UI:** Genshin-Lesbarkeit + Pokémon-Menü-Knuddeligkeit

## Häufige Fehler

| Fehler | Warum schlecht |
|---|---|
| Realistische Proportionen (Kopf = 1/7) | Fühlt sich nicht kindlich an |
| Harte Outlines, viel Schwarz | Wirkt düster |
| Knallbunte Welten ohne Hierarchie | Visuelles Chaos, ADHS-feindlich |
| Ethnische Monokultur | Verfehlt Diversität |
| Große Augen, aber spitze Zähne/Messer | Creepy-Energy |
| Realistische Waffen oder Rüstungen | Mature-Thematik |
