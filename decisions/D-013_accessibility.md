# D-013 — Accessibility-Optionen

**Status:** offen (Validierung in Phase 4)
**Datum:** 2026-06-27
**Owner:** UX Lead

## Problem

Welche Accessibility-Optionen bieten wir an?

## Optionen

### Option A — Drei Profile + Granular-Optionen

- Profile: Standard, Reduziert, Fokus.
- Granular-Optionen für Audio, UI, Hilfen, Tempo.

### Option B — Nur Granular-Optionen

- Viele Einzeloptionen, keine Profile.

### Option C — Nur Standard

- Keine Anpassungsoptionen.

## Bewertungsmaßstab

- Inklusivität
- Komplexität
- Neurodivergenz-Freundlichkeit
- Briefing-Konformität

## Entscheidung

**Option A — Drei Profile + Granular-Optionen.**

## Begründung

- **Briefing-Pflicht:** Drei Reizprofile sind explizit gefordert.
- **Neurodivergenz-Forschung:** Profile machen es einfach, Granular-Optionen ermöglichen Feintuning.
- **UX-Komplexität:** Profile sind einfacher zu kommunizieren als 20 Einzelschalter.

## Erwartete Wirkung

- Breite Inklusivität.
- Geringere Einstiegshürde für Eltern neurodivergenter Kinder.
- Klare UX-Struktur.

## Risiken

- **R-1:** UI-Komplexität.
  - Mitigation: klare Kategorisierung, hilfreiche Tooltips.
- **R-2:** Tests decken nicht alle Profile ab.
  - Mitigation: Phase-4-Tests mit neurodivergenz-erfahrenen Tester:innen.
